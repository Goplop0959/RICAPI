Random Insult

Returns a random insult from our collection of 250+ unique insults.

/random_insult.html
{
  "id": 1,
  "insult": "Your the reason my therapist is well fed!",
  "type": "Savage",
  "source": "local"
}
Savage Tech Sarcastic Corporate Witty Absurd

External API Option

Add ?external=true to fetch from external APIs instead:

/random_insult.html?external=true

Sources: Evil Insult Generator, Matt Bas Insult API

Response Format

All endpoints return JSON with the following structure:

{
  "id": Number,      // Unique identifier
  "type": String,    // Theme/category
  "source": String,  // "local" or "external"
  // Plus one of: "insult", "roast", or "compliment"
}

Usage Examples

JavaScript fetch:

fetch('https://yourusername.github.io/repo/random_insult.html')
  .then(res => res.json())
  .then(data => console.log(data.insult));

cURL:

curl https://yourusername.github.io/repo/random_roast.html

Content Policy

This API maintains a family-friendly profanity level. The following are allowed:

damn, piss, ass, asshole, shit, dick

All other profanity and slurs are strictly prohibited.
